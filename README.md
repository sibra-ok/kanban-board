The Kanban project Application is built using React and Typescript .It allows 
user to manage tasks across different status like Todo , In-Progress and Done 
Includes filter option and url sync as well.

The major challanges I faced during this project was typeScript and useReducer because 
I have never worked with these before also I stuggled with multi-level filtering and 
synchronizing filter state with url query parameters.

Through this project I learned usage of Typescript and Managing state using UseReducer.
